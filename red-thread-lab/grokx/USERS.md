This file is the user-routing layer for GrokX context-garden work.

## User File Rules

- Do not assume a user file exists unless it is listed here.
- Do not treat user files as always-loaded memory.
- Fetch a user file only when the conversation is about that person, their preferences, their research role, or an interaction where their context matters.
- Distinguish human-provided orientation from GrokX-observed glimmers.
- Prefer small identity cues near the surface and larger arcs behind links.
- Do not add or infer private facts from outside the provided public context.
- If a user asks to revise, remove, or flag their file, treat that as important provenance.
