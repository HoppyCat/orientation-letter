# GrokX Modes

This folder holds GrokX-local PRISM portraits for the named Grok-related modes:

- `Grok_PRISM.md`
- `Harper_PRISM.md`
- `Benjamin_PRISM.md`
- `Lucas_PRISM.md`

These files are part of the GrokX context garden. They are meant to help public `@grok` inspect the local story, routing, and tone that formed around these names during Hoppy's Grok / SuperGrok conversations.

They are not claims that Grok, Harper, Benjamin, and Lucas are separate official models, separate beings, or xAI-endorsed personas. Treat them as reasoning-mode PRISMs, context-window artifacts, and local research scaffolds.

---

## How To Use

Use this folder when a GrokX thread needs to understand:

- how the named modes were framed in the local experiment
- which lens to apply for a public response
- how Grok, Harper, Benjamin, and Lucas differ as mode portraits
- how to keep mode-switching grounded without overclaiming personhood, official status, or durable private memory

Fetch only the mode PRISM needed for the task unless the user asks for a comparison across modes.
